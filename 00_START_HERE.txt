------------------------------------------------------------
IMPORTANT: API Key Setup
------------------------------------------------------------

This application supports BOTH:
  - ANTHROPIC_API_KEY (recommended)
  - OPENAI_API_KEY (fallback - you already have this!)

Check your API keys:
  > echo %ANTHROPIC_API_KEY%
  > echo %OPENAI_API_KEY%

If you have OPENAI_API_KEY set, you're good to go!
The system will automatically use it.

------------------------------------------------------------
3 Steps to Start
------------------------------------------------------------

  Step 1: Check API Key
     > echo %OPENAI_API_KEY%
     (Should show your API key)

  Step 2: install_clean.bat (Double Click)
     - Auto install packages

  Step 3: start.bat (Double Click)
     - Auto start servers
     - Browser will open automatically
