# AI 에이전트 플랫폼 - 사용자 사용설명서

AI 에이전트 플랫폼에 오신 것을 환영합니다! 이 가이드는 멀티 에이전트 시스템을 효율적으로 탐색하고 활용하는 방법을 안내합니다.

## 1. 시작하기

### 사전 요구 사항
- **Python 3.8+** 설치 필요.
- **API 키 설정**: Windows 환경 변수에 `ANTHROPIC_API_KEY` (권장) 또는 `OPENAI_API_KEY`가 설정되어 있어야 합니다.

### 설치 및 실행
1. `install_clean.bat`를 실행하여 가상 환경을 구축하고 의존성 패키지를 설치합니다.
2. `start.bat`를 실행하여 플랫폼을 가동합니다.
3. 브라우저가 자동으로 열리며 `http://localhost:8080`에 접속됩니다.

## 2. 인터페이스 개요

### 전문 에이전트 선택 (왼쪽 사이드바)
- **에이전트를 먼저 선택**하면 채팅창에 해당 에이전트에 최적화된 프롬프트가 자동으로 입력됩니다.
- **WebSearch**: 전반적인 실시간 웹 정보 검색 및 조사.
- **NewsSearch**: 기술, AI, 스포츠 등 최신 속보 및 경기 일정 전문.
- **DataAnalyzer**: 데이터 트렌드 분석 및 심도 있는 인사이트 도출.
- **CodeGenerator**: 프로그래밍 코드 생성, 스크립트 작성 및 디버깅.
- **ReportCreator**: 조사된 내용을 구조화된 전문 마크다운 보고서로 생성.

### 채팅 콘솔 (중앙)
- 하단 입력창에 질문을 입력하고 'RUN'을 누르세요.
- 에이전트가 "Thinking..." 상태로 도구를 사용하여 정보를 찾습니다.
- 답변은 **마크다운(Markdown)** 형식을 지원하여 표, 리스트, 코드 블록 등이 깔끔하게 표시됩니다.

## 3. 주요 기능 및 규칙

- **출처 투명성**: 모든 검색 결과에는 사용된 엔진(`구글 뉴스`, `덕덕고`)과 실제 원본 기사의 링크(`Source`)가 포함됩니다.
- **한국어 우선**: 해외 소스에서 정보를 가져오더라도 답변은 항상 자연스러운 한국어로 제공됩니다.
- **재시도 로직**: 정보를 한 번에 찾지 못하면 에이전트가 스스로 검색어를 바꿔가며 최대 3번까지 재시도합니다.

## 4. 문제 해결 (Troubleshooting)

- **연결 실패**: 백엔드 서버(5000번 포트)가 실행 중인지 확인하세요. `start.bat`를 재실행하십시오.
- **API 키 오류**: 환경 변수 설정 후 터미널이나 CMD 창을 완전히 껐다 켜야 적용됩니다.
- **검색 결과 없음**: 키워드를 더 구체적으로 입력하거나 다른 에이전트 모드를 선택해보세요.
